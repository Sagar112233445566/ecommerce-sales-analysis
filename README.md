# 🛒 E-commerce Sales Analysis

Analyzed Brazilian E-commerce data with nearly 1 lakh orders using Python & Pandas.

## Key Findings:
- Total Revenue: R$ 1,35,91,643
- Business grew from R$267 to R$10,10,271 in 14 months
- November 2017 was biggest month (Black Friday effect!)
- 96.9% orders successfully delivered

## Tools Used: 
Python, Pandas, Matplotlib
